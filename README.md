# Projet XANADU - Architecture Cible et Sécurisation du SI

## Vue d'ensemble du projet

XANADU est une entreprise de 50 collaborateurs qui déménage vers la technopole Atlantis et souhaite profiter de cette opportunité pour moderniser et sécuriser son système d'information. L'entreprise va également déployer un bureau distant à Springfield, relié par liaison VPN MPLS.
